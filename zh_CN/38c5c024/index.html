我的日志文件位于何处？

默认情况下，ShotGrid Desktop 和 ShotGrid 集成将其日志文件存储在以下目录中：

Mac

~/Library/Logs/Shotgun/

Windows

%APPDATA%\Shotgun\logs\

Linux

~/.shotgun/logs/

日志文件名采用 tk-<ENGINE>.log 格式。示例包括 tk-desktop.log 或 tk-maya.log。

如果您已将 ShotGrid_HOME 环境变量设置为覆盖用户的缓存位置，那么日志文件将位于：$SHOTGUN_HOME/logs。

注意 : 您还可以从 ShotGrid Desktop 访问此目录。选择一个项目，单击项目名称右侧的向下箭头按钮，然后选择Open Log Folder。
